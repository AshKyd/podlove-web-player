Persistence

Local Storage

Information like the current playtime is persisted in the browser local storage per episode. Episodes meta objects are hashed to create a episode specific hash code: 

{
   "pwp":{
      "750224798":{
         "playtime":25.46358
      },
      "1476730719":{
         "playtime":7.456481
      },
      "-5534497163":{
         "playtime":7.464754
      },
      "-266992392":{
         "playtime":10.949354
      },
      "-2010828542":{
         "playtime":20.966553
      },
      "-1575169385":{
         "playtime":10.450177
      },
      "-970491218":{
         "playtime":8191.483243
      }
   }
}
});

Url

The player configuration can be set via url parameters. This feature is used for deep linking, but can also be used to set custom meta information, like the title:

  • Custom Playtime: persistence.html?playtime=01:33:07
  • Custom Title and Subtitle: persistence.html?title=CustomTitle&subtitle=CustomSubTitle
